## Design Specifications

### Color Palette (Unchanged)

Use these exact colors from the brand:

```
Taupe: #A68C84 (RGB: 166, 140, 132)
Rose Clay: #D9A79B (RGB: 217, 167, 155)
Warm Beige: #F2EDE9 (RGB: 242, 237, 233)
Stone: #C9BCB2 (RGB: 201, 188, 178)
Charcoal: #2B2B2B (RGB: 43, 43, 43)
```

### Typography

**Heading Font**: Cormorant Garamond (or similar elegant serif)
- Used for: "MOYA RITUALS", product names, headlines
- Weights: Light, Regular, Bold

**Body Font**: Inter (or similar clean sans-serif)
- Used for: Body copy, descriptions
- Weights: Regular, Medium

### Logo Usage Guidelines

**Minimum Size**:
- Digital: 150px width
- Print: 1.5 inches width

**Clear Space**:
- Minimum clear space around logo: Height of "R" in "RITUALS"

**Backgrounds**:
- Primary logo works on: White, warm beige, light backgrounds
- Use white version on: Taupe, rose clay, charcoal, dark backgrounds

**Don't**:
- ❌ Stretch or distort logo
- ❌ Change colors
- ❌ Add effects (drop shadows, outlines)
- ❌ Rotate logo
- ❌ Place on busy backgrounds without sufficient contrast
